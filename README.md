一、项目名称
<h2>本草之道</h2>

二、项目背景

在快速发展的现代社会中，中药作为中华民族的瑰宝，其深厚的文化底蕴和独特的健康理念受到越来越多人的关注，涌现出很多中药爱好者，习近平总书记也强调：“要加强中医药传承创新发展。中医药就是适合中国人的‘中国药’”。

然而，中药文化也面临着严峻的挑战。随着现代医学的快速发展，中药文化传承的断层现象越来越明显，年轻一代对中药的历史、理论和疗效缺乏深入了解，对中药文化的认知度和接受度有所下降。同时，“中华药都”等地出现的中药造假现象，使得中药材的辨认难度进一步加大，这一现状不仅损害了中药的声誉，也阻碍了中药文化的传承与发展。

因此，我们决定开发本草之道产品，旨在通过科普中药知识、传承中药传统文化。不管是中药爱好者，还是对中药了解甚少者，都能够轻松学习并享受中药的魅力，从而加强中药传承创新发展。同时我们也希望能潜移默化地提高公众的辨识能力，为有效应对中药市场的假冒伪劣问题做出贡献。

三、产品功能

1.核心功能——药胡啦

1.1功能概述

药胡啦是一款结合中药方剂规则的麻将游戏。通过将传统麻将游戏与中药文化相结合，药胡啦为用户提供了一个有趣、富有教育意义的学习平台，让用户能够在游戏中学习中药方剂知识，感受中药文化的魅力。

1.2游戏规则

药胡啦的游戏规则基于传统麻将游戏进行改编，融入了中药方剂的知识元素。在游戏中，每张麻将牌都代表一味中药。玩家需要通过摸牌、碰杠、胡牌等麻将游戏的基本操作，组合出符合中药方剂规则的牌型。

（1）采药

采药小游戏以“找你妹”为基础，玩家需要在规定的时间内根据页面右边的卡片中药的外观，寻找并采集指定的中药。

（2）炮制

采用横向画卷式的布局，按顺序展示中药炮制的各个环节，每个步骤都配图片和文字介绍，内容是中药起源、中药制作、修制、水制、火制、水火共制。

（3）配伍

配伍小游戏以跳棋为基础，玩家需要移动棋子，同时根据中药配伍的原则“君臣佐使”，选择合适的中药进行搭配。向用户传递“君臣佐使”的中药配伍原则，并让用户了解中药如何搭配使用。

（4）药膳

药膳是中药食疗的推荐，按春夏秋冬四个季节来推荐对应食谱。以瀑布流的布局和图文形式科普，向用户展示中药在日常生活中的运用。通过药膳的学习和实践，用户可以更好地了解中药的养生功效。
